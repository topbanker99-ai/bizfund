# 사장님자금진단 (bizfund)

소상공인·사업주 대상 정책자금·지원금 3분 진단 서비스.

## 구조
- `public/index.html` — 통합 단일 파일. 서버 없이 단독 동작 (더블클릭으로 전체 기능 확인 가능)
- `public/programs-data.js` / `public/match-engine.js` — 데이터·엔진 모듈 버전 (서버 확장 단계용)
- `server.js` — 정적 서빙 + 기업마당 API 동기화 골격 (선택)

## 데이터 정책 (3단 티어)
- 티어1: 수기 검수 데이터만 진단에 투입 (`tier === 1` 필터)
- 티어2: LLM 요건 추출 후보 → 관리자 검수 후 티어1 승격
- 티어3: 기업마당 API 원본 수집분
- `DATA_STATUS = 'draft'` 동안 상단에 검수 전 배너 표시. 공고 원문 대조 후 `verifiedAt` 기입, `'verified'`로 변경.

## 확정된 결정
- 금리 비교: 핀다 크롤링 금지(금소법 리스크) → 정적 비교표 + 아웃링크(금감원·은행연합회·핀다)
- 소진공: 직접 크롤링 대신 기업마당 API 경유

## 2026 기준 수치 (2026-07 검증)
- 최저임금 10,320원 (고용부 고시)
- 4대보험 사업주 요율: 국민연금 4.75% / 건강보험 3.595%(장기요양 13.14% 별도) / 고용보험 약 1.15% / 산재 업종평균 가정 1.47%
- 희망리턴패키지 점포철거비 한도 최대 600만원 (2025.7.11 이후 접수분, 3.3㎡당 20만원)

## 배포
정적 배포(GitHub Pages·Cloudflare Pages·Vercel)는 `public/` 만 올리면 됨.
서버 사용 시: `npm i && BIZINFO_API_KEY=... npm start`
